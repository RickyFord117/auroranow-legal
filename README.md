AuroraNow Legal

This repository hosts the public-facing legal documents for the AuroraNow mobile application.

These documents are served via GitHub Pages to ensure they are publicly accessible, which is a requirement for compliance with the Google Play Store and Apple App Store guidelines.

Documents Included

Privacy Policy: Outlines how AuroraNow collects, uses, and protects user data, including geolocation, anonymous identifiers, and third-party integrations (Supabase, RevenueCat, AdMob).

Terms of Use: Specifies the End User License Agreement (EULA), subscription terms for AuroraNow Pro, and liability disclaimers regarding the unpredictable nature of aurora hunting.
